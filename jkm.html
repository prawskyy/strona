JTM I, II
Ile liczb całkowitych x z przedziału ⟨−2π,2π⟩ spełnia nierówność:
(sin x + 6 cos x + 1) / (3 + cos x) ≤ 2 cos x


JTM II, IIIa

1 / sin(2π/15) + 1 / sin(4π/15) + 1 / sin(8π/15) + 1 / sin(16π/15)


JTM III, III

sinxcosx = sinx + cosx
w przedziale: (13π/4,101π/4)

JTM IV, II
Dane są liczby a,b,c,d w przedziale (0,π), które spełniają równości:
sina + 8sinb = 9sinc + 12sind
cosa + 8cosb = 9cosc + 12cosd
Zakładając, że cos(b-c) ≠ 0, wyznaczyć wartość wyrażenia cos(a-d)/cos(b-c)
